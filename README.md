# cs3100-github-exercise

## Preparation

* Accept the invitation to `espertusnu/cs3100-github-exercise`.
* Open your browser to https://github.com/espertusnu/cs3100-github-exercise.
* Clone the repository in VS Code.

## Process

1. Look over the repository contents.
2. Create an issue either
  * proposing a new recipe
  * proposing a change to a recipe
3. Assign the issue to yourself.
4. Create a branch with a meaningful name, such as `add-pineapple`
  * in VS Code GUI
  * in terminal: `git checkout -b add-pineapple`
  * on github
